# GANs - Redes Generativas Adversarias

Proyecto de prueba de concepto (PoC) explorando GANs con diferentes datasets.

## Contenido

- **GAN.ipynb** - GAN básica entrenada con MNIST (dígitos escritos a mano)
- **Mnist_GAN_ipynb.ipynb** - GAN condicional (cGAN) con MNIST, genera dígitos según la clase indicada
- **blodcells_conditional_gan_keras_implementation.ipynb** - cGAN condicional para generación de imágenes de células sanguíneas

## Dataset de células sanguíneas

https://www.kaggle.com/paultimothymooney/blood-cells

4 tipos de células: Eosinófilos, Linfocitos, Monocitos y Neutrófilos.

## Requisitos

- TensorFlow / Keras
- NumPy
- Matplotlib
- OpenCV (para el notebook de células)

## Notas

Proyecto académico a modo de PoC. Los resultados generados no son de alta calidad pero la arquitectura es funcional.
